# Miro Board Backup

Facilitation canvas for the **Introduction to Nullables** Learning Hour.

## Contents

- `Introduction-to-Nullables-Miro-board.rtb` — the Miro board's native backup.

## Restoring the board

The `.rtb` file is Miro's proprietary backup format and can only be opened by
importing it back into Miro:

1. In Miro, open your dashboard.
2. Create a new board (or open the target team).
3. Use **Import board backup** and select the `.rtb` file.

The backup is not viewable outside Miro — it is stored here for safekeeping and
version control only. It is intentionally kept out of the `docs/` Jekyll site, so
it is not deployed to GitHub Pages.
